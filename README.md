# dev-box

`vagrant` based kubernetes dev-box.

## Installation

    1. Install vagrant via `[official guide](https://www.vagrantup.com/docs/installation/)`
    
    2. Install `virtual box`. (Use as vagrant provider.)

    3. Launch boxes

       ```bash
           vagrant up
       ```
